# deloitte-data-analytics-project
Deloitte Data Analytics Virtual Internship Project - Forage
## Objective
Analyze Daikibo telemetry machine data to identify operational issues and machine downtime patterns.
## Tools Used
- Microsoft Excel
- Tableau
## Files Included
- daikibo-telemetry-data.json.zip (dataset)
- dashboard.png (data visualization dashboard)
- Equality_Table.xlsx (analysis table)
## Key Insights
- Identified machines with the highest downtime
- Analyzed operational performance patterns
- Built a visual dashboard to present insights clearly
